Android Studio 之 Windows 安装

这里记录一下我在Windows 10中安装Android Studio的辛苦历程……

下载

下载地址（国内直达）：下载 Android Studio 和应用工具 - Android 开发者 | Android Developers

这里下载的版本是Android Studio Panda 1 | 2025.3.1 Patch 1

安装

安装过程很简单，基本上是一路next即可。值得注意有两点：选择需要安装的组件环节；安装路径环节

点击查看安装过程

选择安装组件，这里没有勾中“Android Visual Device”

选择一个安装路径，反正我是不把他弄到 C 盘

这里点击“Finish”之后，即将首次打开Android Studio

你和 Android Studio 的第一次

首次打开会询问你是否设置代理和安装SDK等

点击查看详细的过程

这里点击的是“Don't send”

这里点击的是“Cancel”，咱不设置代理，也没有……

这是重点，安装SDK，这里选择的是“Custom”，因为自定义可以选择安装路径

这里的安装路径是D:\Android\Sdk，这也是环境变量ANDROID_HOME和ANDROID_SDK_ROOT的值，据说后者是兼容性写法

不要忘了选择“Accept”

一切顺利，下载完成了。让我感到惊讶的是我应该没有开代理，就这么成功了。一个小小的惊喜。之后点击“Finish”就可以打开Android Studio了
