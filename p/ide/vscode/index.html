快捷键

  • 注释
    下面的三种方式相同：光标所在的某一行；选中的多行（不用选中全部，所覆盖的行都会被注释）
    Ctrl + /
    Ctrl + K + U
    Ctrl + K + C
  • 删除行
    Ctrl + Shift + K：光标所在的某一行；选中的多行（不用选中全部，所覆盖的行都会删除）
  • 多行输入
    Shift + Alt：同时按下这两个键，鼠标选择即可
  • 进入新的一行输入
    Ctrl + Enter：在光标所在行的下方插入以新行，并且光标进入创建的新行
  • 向上或者向下复制当前行
    Shift + Alt + 上键/下键：在光标所在行的下方插入以新行，并且光标进入创建的新行

Prettier - Code formatter

格式化代码必备插件。官网地址：Prettier · Opinionated Code Formatter、在线预览效果：Prettier Online 。其实VSCode自带也不错，但是支持文件类型有限（尤其是.vue文件），所以选择了她。Prettier有很多配置项，详细的可以看一下官网：Options · Prettier，有些配置项是有代码示例的。

下面是我们的配置项：

{
    "arrowParens":"always",
    "bracketSameLine":true,
    "bracketSpacing":true,
    "embeddedLanguageFormatting":"auto",
    "endOfLine":"lf",
    "htmlWhitespaceSensitivity":"css",
    "jsxSingleQuote":false,
    "printWidth":1024,
    "proseWrap":"preserve",
    "quoteProps":"consistent",
    "requirePragma":false,
    "semi":true,
    "singleAttributePerLine":false,
    "singleQuote": false,
    "tabWidth": 4,
    "trailingComma": "none",
    "useTabs":false,
    "vueIndentScriptAndStyle":true
}

你可以将上面的配置项放到一个JSON文件中，并添加到Prettier: Config Path配置项中
