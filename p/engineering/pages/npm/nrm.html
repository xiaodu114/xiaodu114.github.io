nrm

nrm：NPM registry manager, fast switch between different registries: npm, cnpm, nj, taobao

GitHub地址：GitHub - Pana/nrm: NPM registry manager, fast switch between different registries: npm, cnpm, nj, taobao

安装

npm install -g nrm

使用

帮助

nrm -h

查看可用的 registry

nrm ls

切换 registry

nrm use taobao
