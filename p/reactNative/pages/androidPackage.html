生成私有签名密钥

可以使用keytool生成私有签名密钥。keytool是啥？你安装了吗？既然都看到了这里，我觉得已经安装了，你看这里：

来吧，进入android/app目录并打开终端，输入下面的命令：

keytool -genkeypair -v -storetype PKCS12 -keystore myapp001-android-key.keystore -alias myapp001-android-key-alias -keyalg RSA -keysize 2048 -validity 10000

因为官方文档后面的介绍是将密钥文件放到android/app目录下，所以这里直接再该目录下生成

  • myapp001-android-key.keystore：密钥文件名
  • myapp001-android-key-alias：别名

APK

如果不是第一次执行，建议先清理一下：

./gradlew clean

终端进入项目下android目录，执行下面的命令：

./gradlew assembleRelease

APK在这里哦！你发现了吗？android/app/build/outputs/apk/release/app-release.apk

npx react-native build-android --tasks assembleRelease

这个是后补的。打包AAB时，才知道有这种方式（请看下文），于是查一下是否有类似的命令来打包APK，真有：

npx react-native build-android --tasks assembleRelease

AAB

如果不是第一次执行，建议先清理一下：

./gradlew clean

终端进入项目下android目录，执行下面的命令：

./gradlew bundleRelease

AAB在这里哦！你发现了吗？android/app/build/outputs/bundle/release/app-release.aab

npx react-native build-android --mode=release

配置打包文件名称

无论是APK还是AAB，文件名称都是app-release，一点都不高大上……

怎么弄呢？还是得修改android\app\build.gradle这个文件，在上面内容的基础上添加下面的内容：

APK 文件名

android {
    ...
    defaultConfig {   
    }
    signingConfigs {   
    }
    buildTypes {    
    }
    // 这里，这里，这里，APK 命名规则
    applicationVariants.configureEach { variant ->
        variant.outputs.configureEach { output ->
            if (variant.buildType.name == "release") {
                def versionName = variant.versionName
                def date = new Date().format("yyyyMMdd", TimeZone.getTimeZone("Asia/Shanghai"))
                output.outputFileName = "MyApp001-v${versionName}-${date}.apk"
            }
        }
    }
}

AAB 文件名

android {
}

dependencies {
}

// 这里，这里，这里，AAB 命名规则
tasks.whenTaskAdded { task ->
    if (task.name == "bundleRelease") {
        task.doLast {
            def aabDir = file("${project.buildDir}/outputs/bundle/release/")
            def date = new Date().format("yyyyMMdd", TimeZone.getTimeZone("Asia/Shanghai"))
            def versionName = android.defaultConfig.versionName
            
            aabDir.listFiles()?.each { file ->
                if (file.name.endsWith('.aab')) {
                    file.renameTo(new File(file.parent, "MyApp001-v${versionName}-${date}.aab"))
                }
            }
        }
    }
}
