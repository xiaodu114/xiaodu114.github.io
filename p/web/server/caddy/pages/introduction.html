下载

下载地址：Download Caddy

根据自己的平台下载即可。这里的测试环境是windows，下载的是一个绿色版的.exe文件。

安装

虽然是个.exe文件，但是不用安装。直接扔到一个文件夹，这里是D:\ProgramOther\web-servers\caddy-server，为了后面方便使用，将他弄到了环境变量中。之后，你可以在任意目录的cmd或者powershell输入caddy测试一下；如果你忘记了他的位置，还可以使用下面的命令看一下

#   添加到环境变量之后，你可以使用下面的命令查找 caddy.exe 所在位置
where.exe caddy
