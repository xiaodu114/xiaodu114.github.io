OpenIM 安装

部署环境是Windows 10 + Docker Desktop。如果想看安装Docker Desktop等更多内容，可以移步这里：Docker - xiaodu114.github.io

参考地址：docker部署 | OpenIM Docs

克隆仓库

git clone https://github.com/openimsdk/openim-docker

启动服务

点击查看 Docker 中镜像和容器 

docker compose up -d

停止服务

docker compose down

查看日志

docker compose logs -f openim-server openim-chat
